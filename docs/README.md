# About
Coin Clicker is a web-based idle game inspired by Cookie Clicker.

# Download
You can get any stable release of the game from the Releases tab on the right. Simply pick the version you want, and click 'Source code (zip)'  
Or, you can play the game [here.](https://xatra1.github.io/coin-clicker-rewrite)
